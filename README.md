# Loan-Default-Prediction
This project analyzes retail loan data to identify key indicators of borrower default. By employing Logistic Regression, I built a classification model to assess risk and provide actionable insights for credit underwriting.
Loan Default Risk Analysis

This project analyzes a financial lending dataset containing over 38,000 loans to understand the determinants of loan default.

Techniques used:

Data visualization
Logistic regression modeling
